# Literature Review Workflow

## Steps
1. Define research question.
2. Use literature-scout agents from domain repos.
3. Ingest into paper.schema.json.
4. Build evidence-table.schema.json.
5. Synthesize into claim.schema.json.
6. Run reproducibility-review.

## Agent Support
- literature-scout (domain specific)
- construct-mapper (psych)
- neuro-literature-scout (neuro)

## Output
Evidence table + claim graph ready for research-intelligence-os pipelines.

Useful for systematic reviews and personal canon building from research.
