# Neuroscience Research Pack

## Purpose
Reusable pack for neuroscience research intelligence within the Mind Intelligence Systems portfolio.

Focus: Literature, datasets (OpenNeuro, DANDI), BIDS/NWB standards, EEG/MEG/fMRI pipelines (MNE-Python, PyNWB), reproducibility, open science.

## Latest Best Tech (2026)
- BIDS 1.10+ specification for standardized data organization.
- NWB 2.7 for neurodata storage and sharing.
- MNE-Python 1.8+ for EEG/MEG analysis pipelines.
- SpikeInterface for spike sorting.
- DANDI for NWB data archiving.
- OpenNeuro for public datasets.
- Preregistration and Registered Reports for robust science.
- Reproducibility with containerization (Docker, Singularity) and workflow managers (Snakemake, Nextflow).

## How People Experience It
Researchers start with literature scouting, move to dataset discovery, build reproducible pipelines, generate QC reports, and synthesize evidence into claim graphs.

Daily/Weekly: Ingest papers or datasets, run validation, execute pipelines, review reproducibility.
