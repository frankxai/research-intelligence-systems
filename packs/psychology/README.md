# Psychology Research Pack

## Purpose
Reusable pack for psychology research intelligence.

Focus: Constructs, measurement, psychometrics, qualitative analysis, behavioral experiments, evidence synthesis, reproducibility.

## Latest Best Tech (2026)
- Preregistration via OSF or AsPredicted.
- Registered Reports for publication.
- Modern psychometrics: pingouin, psychopy, statsmodels, lavaan (R).
- Qualitative: MAXQDA alternatives or Python (e.g., for thematic analysis with LLMs + human oversight).
- Surveys: Qualtrics API or open source like LimeSurvey.
- Evidence synthesis: meta-analysis with metafor or Python equivalents.
- Open science: FAIR principles, data sharing via OSF.

## How People Experience It
Psychologists map constructs, design studies, evaluate measures, synthesize literature, design experiments.

Use with agentic-mind-os for personal reflection on research practices.

## Agent Exploration Paths
Agents: construct-mapper, literature-scout, methods-critic, psychometrics-agent, qualitative-analysis-agent, behavioral-experiment-designer.

Prompt example: "As construct-mapper, map 'self-efficacy' using human-mind models (belief, motivation, identity). Output to construct.schema.json. Link to psychologyresearch.yaml."

Use workflows/construct-to-measurement.md, literature-to-evidence-table.md.

## Usefulness
- Improves rigor in psychological science.
- Supports reproducibility crisis solutions.
- Integrates with neuroscience for cognitive psych.
- Agentic workflows reduce manual labor.
- Benefits researchers, students, funders.
