# Cognitive Science Pack

## Purpose
For cognitive science research: perception, attention, memory, decision making, consciousness studies.

## Latest Best Tech
- Computational modeling with Python (e.g., using PyMC for Bayesian, or custom).
- Eye tracking analysis with PyGaze or PsychoPy.
- fMRI/EEG integration with MNE + Nilearn.
- Preregistration, power analysis with pingouin or G*Power.
- Registered Reports.

## Experience
Integrates with psychology and neuroscience packs.
Agents use human-mind models for attention, memory, decision-making, consciousness.

## Usefulness
